Project COBOL
=============
- Examples for OpenCobol (Gnu Cobol)
- Examples for AS400, PF, LF, DSPF
- Examples for ILE Cobol, ILE CL

### AS400 ILE COBOL
   * ILE COBOL 
   * ILE CL 
   * Physical files
   * Logical files
   * Display files
   
### OpenCOBOL/GNUCobol
   * Hello Cobol
   * Conditions	
   * Database	
   * Date
   * Games
   * Loops	
   * Memory	
   * Random	
   * SQLite	
   * Sort	
   * String
   * Struct 
   
#### Compilation with GnuCobol compiler

```sh
cobc -x -W -Wall -O2 HelloWorld.cbl
```
